# Ai-handwritten-digit-detector

This project detects handwritten digits using Machine Learning.

## 📌 Technologies Used
- Python
- OpenCV
- NumPy
- TensorFlow / Keras

## 📂 Project Files
- train.py → Model training
- predict.py → Predict digit
- predict_draw.py → Draw and predict
- digit_cnn.h5 → Trained model

## ▶️ How to Run
pip install -r requirements.txt
python predict.py

## ⚠️ Note
Make sure all dependencies are installed before running.
